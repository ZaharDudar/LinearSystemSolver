# Библиотека для решения СЛАУ

Типы хранения матриц
---
### Реализованны следующие типы:

* `Matrix` - стандартный класс, хранящий плотную матрицу
* `CSRMatrix` - класс, в котором реализован формат хранения разреженной матрицы CSR.
